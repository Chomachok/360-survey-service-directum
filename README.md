## 📧 Локальный SMTP-сервер для разработки (Mailpit)

Для тестирования отправки писем (приглашений на опрос) мы используем **Mailpit** – локальный SMTP-сервер с веб-интерфейсом. Он перехватывает все письма, отправленные приложением, и позволяет просматривать их в браузере. Это безопасно, бесплатно и не требует регистрации.

### 🚀 Запуск Mailpit

1. **Убедитесь, что Docker установлен и запущен** на вашем компьютере.
2. Перейдите в корневую папку проекта (где лежит `docker-compose.yml`).
3. Выполните команду:
   ```bash
   docker-compose up -d
